Understanding the Derivative, Partial Derivative, and Gradient

What is a Derivative?

A derivative measures how a function changes as its input changes. For example, it tells us the rate of change of one quantity with respect to another.

The derivative of a function at a given point represents the slope of the tangent line to the function’s graph at that point.

Equation of the Curve:

Let the curve be defined as:

f(x)=x3−3x2+2x+1.f(x) = x^3 - 3x^2 + 2x + 1.f(x)=x3−3x2+2x+1.

Derivative of the Function:

To find the slope of the tangent line, compute the derivative (f’(x)):

f′(x)=ddx(x3−3x2+2x+1).f'(x) = \frac{d}{dx}(x^3 - 3x^2 + 2x + 1).f′(x)=dxd​(x3−3x2+2x+1).

Using the power rule:

f′(x)=3x2−6x+2.f'(x) = 3x^2 - 6x + 2.f′(x)=3x2−6x+2.

Slope(Derivative) at a Specific Point:

Suppose we want to find the slope of the tangent at (x = 2). Substitute (x = 2) into (f’(x)):

f′(2)=3(2)2−6(2)+2=2f'(2) = 3(2)^2 - 6(2) + 2 = 2f′(2)=3(2)2−6(2)+2=2

The slope at x = 2 is 2

What is a Gradient?

The gradient is just a fancy term used when you take the derivative of a function with respect to all of its variables. In essence, it is a collection of all the partial derivatives of a function.

  • For a function f(x,y,z)f(x, y, z)f(x,y,z), the gradient vector is written as:
    ∇f=[∂f∂x∂f∂y∂f∂z]\nabla f = \begin{bmatrix} \frac{\partial f}{\partial x} \\ \frac{\partial f}{\partial y} \\ \frac{\partial f}{\partial z} \end{bmatrix}∇f=⎣⎡​∂x∂f​∂y∂f​∂z∂f​​⎦⎤​

  • The gradient points in the direction of the steepest increase of the function and is widely used in optimization problems, such as in machine learning.

For example If f(x,y)=x2+y3f(x, y) = x^2 + y^3f(x,y)=x2+y3 then:
∇f=[2x3y2]\nabla f = \begin{bmatrix} 2x \\ 3y^2 \end{bmatrix}∇f=[2x3y2​]

Example to show What Partial Derivatives do

  • Remember: the rate of change of the function with respect to one variable while keeping others constant(frozen) is called a partial derivative. This is done to see how one variable affects the output of the function when all other variables are constant/frozen.

Step 1: Initial Output

We have the function:

f(x,y)=x2+y2.f(x, y) = x^2 + y^2.f(x,y)=x2+y2.

Suppose (x = 3) and (y = 4). The initial output of f(x, y) is:

f(3,4)=32+42=9+16=25.f(3, 4) = 3^2 + 4^2 = 9 + 16 = 25.f(3,4)=32+42=9+16=25.

Step 2: Small Changes in (x) and (y)

Let x increase by Δx=0.1 and y increase by Δy=0.2. The new values are:\text{Let } x \text{ increase by } \Delta x = 0.1 \text{ and } y \text{ increase by } \Delta y = 0.2 \text{. The new values are:}Let x increase by Δx=0.1 and y increase by Δy=0.2. The new values are:
x=3+0.1=3.1,y=4+0.2=4.2.x = 3 + 0.1 = 3.1, \quad y = 4 + 0.2 = 4.2.x=3+0.1=3.1,y=4+0.2=4.2.

Step 3: New Output

The new output of f(x,y) is:\text{The new output of } f(x, y) \text{ is:}The new output of f(x,y) is:

f(3.1,4.2)=(3.1)2+(4.2)2=9.61+17.64=27.25.f(3.1, 4.2) = (3.1)^2 + (4.2)^2 = 9.61 + 17.64 = 27.25.f(3.1,4.2)=(3.1)2+(4.2)2=9.61+17.64=27.25.

Step 4: Change in Output

The change in the output of (f(x, y)) is:

Δf=f(3.1,4.2)−f(3,4)=27.25−25=2.25.\Delta f = f(3.1, 4.2) - f(3, 4) = 27.25 - 25 = 2.25.Δf=f(3.1,4.2)−f(3,4)=27.25−25=2.25.

Step 5: Approximation Using Gradients

original function: f(x,y)=x2+y2.f(x, y) = x^2 + y^2.f(x,y)=x2+y2.
x increases by Δx=0.1 and y increases by Δy=0.2\text{} x \text{ increases by } \Delta x = 0.1 \text{ and } y \text{ increases by } \Delta y = 0.2 x increases by Δx=0.1 and y increases by Δy=0.2

Δf≈∂f∂x⋅Δx+∂f∂y⋅Δy, \Delta f \approx \frac{\partial f}{\partial x} \cdot \Delta x + \frac{\partial f}{\partial y} \cdot \Delta y, Δf≈∂x∂f​⋅Δx+∂y∂f​⋅Δy,

where:

∂f∂x=2x,∂f∂y=2y. \frac{\partial f}{\partial x} = 2x, \quad \frac{\partial f}{\partial y} = 2y. ∂x∂f​=2x,∂y∂f​=2y.

At (x = 3) and (y = 4), we calculate:

∂f∂x=2(3)=6,∂f∂y=2(4)=8. \frac{\partial f}{\partial x} = 2(3) = 6, \quad \frac{\partial f}{\partial y} = 2(4) = 8. ∂x∂f​=2(3)=6,∂y∂f​=2(4)=8.

Using the gradient, the approximate change in (f) is:

Δf≈∂f∂x⋅Δx+∂f∂y⋅Δy, \Delta f \approx \frac{\partial f}{\partial x} \cdot \Delta x + \frac{\partial f}{\partial y} \cdot \Delta y, Δf≈∂x∂f​⋅Δx+∂y∂f​⋅Δy,

What This Shows

This example demonstrates that partial derivatives reveal how changes in each input affect the function’s output:

  1. For x, the partial derivative is 6, meaning the output changes 6 times the increment in x. A 0.1 increase in x results in aΔf=6⋅Δx=6⋅0.1=0.6 \Delta f = 6 \cdot \Delta x = 6 \cdot 0.1 = 0.6 Δf=6⋅Δx=6⋅0.1=0.6 change in the output of the function.
    1. For y, the partial derivative is 8, meaning the output changes 8 times the increment in y. A 0.2 increase in y results in aΔf=8⋅Δy=8⋅0.2=1.6 \Delta f = 8 \cdot \Delta y = 8 \cdot 0.2 = 1.6 Δf=8⋅Δy=8⋅0.2=1.6 change in the output of the function.

Partial derivatives provide a clear rate of change for each variable.
